## About the project
* Just I testing how to create a project base with Laravel Sail.

## How to
* git clone https://github.com/kekosoftware/test-sail.git
* cd test-sail
* Start Sail in detached mode: ./vendor/bin/sail up -d
* Test: http://localhost
* Stop Sail: ./vendor/bin/sail down
